Graduates Course Registration Schedule（2017-2018 Spring Semester ）

Note: Only after doing personal curriculum, can graduates register courses.  
 
Pre-registration                         2018-01-02 (Tues.) 13:00 ～ 2018-01-05 (Fri.) 11:00      
Students should prioritize the selected courses (except make-up courses).
The system will run a random draw. Please check the result of draw after 12:00, Jan5.
Make-up Course given by Department of Mathematical Sciences，Department of Foreign Languages and Literatures are counted in course capacity. When the class is full, any student cannot be enrolled in.
 
Course Registration                  2018-01-05 (Fri.) 13:00 ～ 2018-01-06 (Sat.) 16:00
First come, first served. No more applicants will be accepted when course capacity is up to the maximum.
 
Adjustment stage I                     2018-2-28 (Wed.) 13:00 ～ 2018-03-05 (Mon.) 8:00
First come, first served. No more applicants will be accepted when course capacity is up to the maximum. All courses can be canceled during the period.
 
Adjustment stage II                    2018-03-05 (Mon.) 13:00 ～ 2018-03-09 (Fri.) 16:00      
Students only can delete courses and cannot add any courses during this term.
The deleted courses will not be listed on transcripts.
 
Course Drop Periods
The dropped courses (except those begin from or after 9th academic week) will be marked with“W”（Withdrew）on transcript.
Course Drop I                           2018-04-02 (Mon.) 8:00 ～ 2018-04-08 (Sun.) 16:00      
Course Drop II                          2018-04-29 (Sun.) 8:00 ～ 2018-05-11 (Fri.) 16:00  
Students cannot drop: Courses finished in first 8 weeks
The dropped courses (except those begin from or after 9th academic week) will be marked with “W”（Withdrew）on transcript.
 
Tips for adding/dropping course
 
Enter course registration system by logging into: http://academic.tsinghua.edu.cn. You can select English /Chinese language to access to course registration system. The system supports Win7, IE 9-11 browser. Other system or web browser may result in unknown error and failure on adding/deleting course. If problems found on personal computer, please go to the university lab on main building to do course-registration.
 
If you are off campus, you should set a VPN client to visit the intranet to do any course registration.
