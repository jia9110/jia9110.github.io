Undergraduates Course Registration Calendar (2017-2018 Spring Semester)

 

选课系统分为中、英文两种语言，同学们可选择其中一种语言进入选课系统。

 

选课系统适用于 win7 操作系统， IE 浏览器 9-11 版本均可。 使用其他版本的操作系统或浏览器进行选退课操作，有可能出现异常情况导致选退课失败。若出现同学自己的电脑环境不支持选课系统的情况，请及时到中央主楼开放实验室机房进行选课操作。

 

使用校外网络选课的同学，请务必按照附件中的操作步骤，安装 VPN 客户端后方可进行选课操作。

关于校外选课的网络操作说明.pdf

 

选课登录入口：清华大学教学门户（   http://academic.tsinghua.edu.cn   ）

电话咨询： 选课 问题，请致电注册中心 62794721，62788616转16、15

1、本科生正选、补退选第一阶段，实行Waiting List方案。详见 教务公告 《 关于在本科生选课系统实行Waiting List方案 通知 》 。

 

2、 补退选第一阶段结束后，所有本科生体育课、 部分外文系课程（课程号为0064或1064开头） 不允许删选或退课。

补退选（第二阶段）： 2018 年3 月5 日 ( 星期一 ) 13:00 -- 3 月 9 日 ( 星期五 ) 16:00

  *  体育课、部分外文系课程（课程号为0064或1064开头）不可选，亦不可删

选择课程以 P/F 记载 （仅限 2015 年及以后入学的本科生）

 

         2018 年4 月 2 日 ( 星期一 ) 8:00 -- 4 月8 日 ( 星期日 ) 16:00

    *  在已选的任选课程中可选择一门，其成绩记为 P （ Passing ） / F （ Failure ），无绩点

 

注意事项：

  • 选课系统中的课程属性与选课属性是两个不同的概念，课程属性是以学生主修专业培养方案为依据，选课属性是以当学期的教学计划为依据（非当学期教学计划中的课程，其选课属性均为任选）。选择课程以P/F记载，是在课程属性为“任选”的课程中进行选择。
  • 建议同学们在选择P/F记载课程前，务必先咨询本院系教务老师，确认没有后续影响后再做选择。
  • 辅修、双学位的学生应注意，请先咨询辅双院系能否将辅双课程选为P/F记载。
  • “P/F记载选择”为期五天，网选结束后不可更改。
  • 已选择P/F记载的课程，如在退课阶段操作退课，则本学期的P/F选择权作废。
  • 若要选择多学期课程以P/F记载，请在课程开课的最后一个学期进行选择。
