Undergraduates Course Registration Calendar(2017-2018 Spring Semester)
【分类: 综合办公室】  【发布时间: 2017-12-12】  【访问计数:755】
   【关闭窗口】
 

 

Pre-registration (registration application)   Dec. 15 (Fri.) 13:00 ～ Dec. 18 (Mon.) 16:00, 2017

 

Students should prioritize the selected courses (except make-up courses). A random draw will be run when pre-registration ends. Please check the results of the draw before Course Registration begins.

Students are suggested to register make-up courses during this term. All make-up courses are available for registration.

Note: Make-up Courses given by Department of Mathematical Sciences and Department of Foreign Languages and Literatures are counted in course capacity. When the class is full, any student cannot be enrolled in.

All make-up PE courses only can be registered during the two periods: Pre-registration and Course Registration stage.

 

Course Registration   Dec. 22 (Fri.) 13:00 ～ Dec. 25 (Mon.) 16:00, 2017

 

Lab-course Registration I   Dec. 29 (Fri.) 13:00, 2017 ～ Jan. 4 (Thur.) 16:00, 2018

 

Adjustment stage I   Feb. 28 (Wed.) 13:00 ～ Mar. 5 (Mon.) 8:00, 2018

 

Adjustment stage II   Mar. 5 (Mon.) 13:00 ～ Mar. 9 (Fri.) 16:00, 2018

Students only can delete courses and cannot add any courses during this term.

Note: students cannot delete PE courses and courses whose number begin with 0064 /1064 (general language courses given by Department of Foreign Languages and Literatures). The deleted courses will not be listed on transcripts.

 

Lab-course Registration II   Feb. 28 (Wed.) 13:00 ～ Mar. 5 (Mon.) 8:00, 2018

 

P/F Marking Scheme Course Selection   Apr. 2 (Mon.) 8:00 ～ Apr. 8 (Sun.) 16:00, 2018

 

Students can select one course from their registered electives to set it as a P （ Passing ） / F （ Failure ） marking scheme course. No points given to the P/F course.

 

Course Drop Periods

 

Course Drop I   Apr. 2 (Mon.) 8:00 ～ Apr. 8 (Sun.) 16:00, 2018

 

Note: PE Courses and Courses whose number begin with 0064/1064 (general language courses given by Department of Foreign Languages and Literatures) cannot be dropped. The dropped courses will be marked with “ W ”（ Withdrew ） on transcript.

 

Course Drop II   Apr. 29 (Sun.) 8:00 ～ May. 11 (Fri.) 16:00, 2018

Students cannot drop: Courses finished in first 8 weeks, PE courses and courses whose number begin with 0064/1064 (general language courses given by Department of Foreign Languages and Literatures).   The dropped course will be marked with “ W ”（ Withdrew ） on transcript.

 

Please note the changes on course registration:

 

A method of waitlist will be used for course registration. Please refer to Academic Announcements to read more about waiting list: http://announce.cic.tsinghua.edu.cn/node/26739

 

When Adjustment stage I ends, students cannot drop any PE courses and courses whose number begin with 0064/1064 (general language courses given by Department of Foreign Languages and Literatures).

 

Tips for adding/dropping course:

 

Enter course registration system by logging into: http://academic.tsinghua.edu.cn. You can select English /Chinese language to access to course registration system. The system supports Win7, IE 9-11 browser. Other system or web browser will result in unknown error and failure on adding/deleting course, if problem found on personal computer, please go to the university lab on main building to do course-registration.

 

If you are off campus, you should set a VPN client to visit the intranet to do any course registration.

 

Course Registration service: Registrar ’ s Office,    62788616 extention16 /15
