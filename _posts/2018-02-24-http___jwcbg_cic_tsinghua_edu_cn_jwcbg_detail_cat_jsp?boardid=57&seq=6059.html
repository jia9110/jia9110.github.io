关于紫荆 C 楼部分教室变更的通知

 

因我校紫荆 C 楼部分教室出现变故，请通知你院 ( 系 ) 有关任课教师及上课班级学生，下学期（ 2017-2018 春）在紫荆 C 楼教室上课的部分课程，地点变更如下：

原紫荆 C 楼 308 教室的课，调至紫荆 C 楼 201 教室上课；

原紫荆 C 楼 316 教室的课，调至紫荆 C 楼 204 教室上课；

原紫荆 C 楼 317 教室的课，调至紫荆 C 楼 202 教室上课；

原紫荆 C 楼 320 教室的课，调至紫荆 C 楼 205 教室上课。

 

       因教室的变更，给大家带来的不便，我们深表歉意，谢谢大家的理解与支持，并请大家相互转告！
